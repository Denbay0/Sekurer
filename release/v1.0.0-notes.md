# Sekurer v1.0.0

Backend, worker, AI call processing, Flutter Android/iOS ready, smoke-test passed.

## Highlights
- FastAPI backend with Postgres, Redis, MinIO startup checks.
- Celery worker status endpoint: `/worker/status` and `/api/v1/worker/status`.
- Call upload pipeline with AI transcription/analysis flow and task extraction.
- Safe client-facing errors and backend/automation logging.
- Full smoke automation: register, login, upload audio, worker processing, tasks check.
- Flutter Android release APK and App Bundle build verified.
- Flutter iOS project files generated for macOS/Xcode builds.
